# Игра "Проклятие Древнего Леса"

Выполнил: Шафиев Марсель

Группа: М3О-137БВ-25

## Описание игры

RPG игра, в которой игрок исследует древний лес, собирает артефакты и сражается с врагами. Цель - победить финального босса и спасти лес от тьмы.

## Сюжет игры

**Начало:** Вы - молодой искатель приключений, получивший письмо от старого друга. В письме он пишет о странных явлениях в Древнем Лесу и просит о помощи, но затем связь обрывается.

**Развитие:** Отправляясь на его поиски, вы встречаете:

1. **Лесные гоблины** - первая опасность в заброшенной хижине на опушке леса
2. **Речные твари** - обитатели бурной реки, преграждающей путь
3. **Каменные големы** - древние стражи храма, построенного забытой цивилизацией
4. **Пещерный тролль** - ужасное существо, охраняющее сокровища в глубинах пещеры

**Кульминация:** По мере продвижения вы находите записку друга, из которого узнаёте, что **Теневой Волк** захватил лес. Именно он напал на друга героя и теперь угрожает всему королевству.

**Развязка:** Чтобы победить Теневого Волка, нужно собрать шесть древних артефактов, рассеянных по лесу. Каждый артефакт дает силу, необходимую для финальной битвы.

## Испытания

1. **Заброшенная хижина** - Поиск следов Элрика, встреча с гоблинами
2. **Бурная река** - Переправа через опасную реку, встреча с речными тварями
3. **Древний храм** - Исследование храма, битва с големами-стражами
4. **Пещера сокровищ** - Поиск артефактов, схватка с пещерным троллем

## Основные возможности игры

- **Вход и регистрация** по логину и паролю
- **Сохранение прогресса**
- **4 испытания** с выбором (каждый выбор влияет на развитие)
- **Сбор артефактов** (всего 6 штук)
- **Система лечения** зельями (ограниченный запас)
- **Финальный босс** с повышенной сложностью

## Особенности системы

- Если игрок умирает - артефакты возвращаются в общий файл
- Если все артефакты разобраны - они автоматически восстанавливаются
- Прогресс сохраняется автоматически после каждого испытания
- Враги сильные - нужно стратегически использовать лечение


## Файлы проекта

- **main.py** - запуск игры, меню, регистрация и вход
- **classes.py** - классы игрока и босса
- **data_manager.py** - работа с файлами сохранений
- **adventure.py** - испытания и боевая система
- **users.txt** - логины и пароли игроков
- **items.txt** - общий список артефактов
- **save_логин.txt** - профили игроков (создаются автоматически)

## Боевая система

В бою доступны три действия:
1. **Атаковать** - нанести урон противнику
2. **Лечиться** - использовать зелье лечения (+30 HP)
3. **Особый удар** - мощная атака (требует артефакты)

## Артефакты для сбора

1. **Меч Луны** - увеличивает атаку
2. **Щит Солнца** - увеличивает здоровье
3. **Кольцо Ветров** - увеличивает атаку
4. **Амулет Земли** - увеличивает здоровье
5. **Плащ Теней** - увеличивает атаку
6. **Посох Магов** - увеличивает здоровье
